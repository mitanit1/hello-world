3 trạng thái GIT
- modified: Đã sửa, chưa lưu
- staged: chuẩn bị lưu local
- commited: Lưu trong db Local

3 khu vực
- working tree: Chứa bản sao phiên bản cụ thể của dự án
- staging: File nằm trong thư mục git, chứa thông tin sẽ commit
- Git directory: Nơi Git lưu database các file Metadata


git init
git clone
git add "filename", git add .
git commit -m "comment"
git log, git log --oneline
git checkout #hag -- "file", git checkout #hag .
git status
git reset --soft HEAD~1, git reset --hard HEAD~1
git reset -- "file name", git reset -- .
git restore --staged "file name", git restore --staged .

git branch.
